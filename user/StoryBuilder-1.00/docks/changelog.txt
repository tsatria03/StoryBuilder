New in 1.0.
Addressed an issue where returning certain functions with the return command would crash the entire program.
Improved the return command. It can now return actual values such as strings, numbers, and calls to other funcions.
Added the ability to apply aditional settings for controling cancelation types to the writemode command.
Fixed major comparison issues with nested conditions.
Added the ability to configure character restriction settings for typing in input boxes with the new writemode command.
Fixed some bugs with executing various nested loop types.
Added the ability to write nested conditions and loops.
Added the ability to generate various types of passwords with the new genpass command.
Added whildcard support to playing sounds in stories.
Fixed some major issues with managing variables for while loops.
Fixed some major issues with initializing while loops.
Fixed some bugs with interacting in menus.
Fixed some bugs with calling functions
Added the ability to create local variables for writing functions.
Added the ability to create arguments for writing functions.
Added the ability to create functions and while loops in your story.
Fixed an issue where pressing escape in menus and input boxes would not break out of them propperly.
Added the ability to toggle hearing dialog sounds with the new textsound command. You can type this in your stories to disable them at certain points.
Added a large collection of commands to control various asspects of your story. This includes things like dialogs, functions, loops, controls, audio and game systems, variable management, and much much more.
Added a comprehencive readme file to the docks folder that explains the core features of this mini story scripting language.
